# whack-a-mole
A whack-a-mole game with a timer, sound and high score tracker
https://jacobcons.github.io/whack-a-mole/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
